# Linear Regression with TensorFlow

This directory contains a Python script (`linear_regression_tf.py`) that implements a simple linear regression model using TensorFlow.

## Requirements

- Python 3.x
- TensorFlow 2.x

## Installation

To install the required dependencies, run:

```bash
pip install tensorflow
```

## Usage

To run the linear regression script:

```bash
python linear_regression_tf.py
```

The script will:
1. Generate sample data for linear regression
2. Create and train a simple neural network model
3. Display the model summary and training progress
4. Make a sample prediction
5. Save the trained model as 'linear_regression_model.h5'

Note: If you encounter any issues with TensorFlow installation, please refer to the official TensorFlow installation guide: https://www.tensorflow.org/install
