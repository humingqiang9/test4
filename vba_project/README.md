# Excel Sheet Formatting VBA Macro

This project contains a VBA macro that formats an Excel sheet with common formatting options.

## Macro Features

- Formats header row with bold font, larger size, and background color
- Adds borders to cells
- Auto-fits columns for better readability
- Centers align all data
- Adds a title to the sheet

## How to Use

1. Open Excel and press `Alt + F11` to open the VBA editor
2. Insert a new module (`Insert > Module`)
3. Copy and paste the code from `format_macro.vba` into the module
4. Close the VBA editor
5. Run the macro by pressing `Alt + F8`, selecting "FormatExcelSheet", and clicking "Run"

## Requirements

- Microsoft Excel
- VBA support enabled

The macro is designed to work with data in the range A1:E100. You can modify the ranges in the code as needed for your specific worksheet.
